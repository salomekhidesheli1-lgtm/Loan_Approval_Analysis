# Loan Approval Analysis Dashboard

The loan approval dataset is a collection of financial records and associated information used to determine the eligibility of individuals or organizations for obtaining loans from a lending institution. It includes various factors such as CIBIL score, income, employment status, loan term, loan amount, assets value, and loan status.

This end-to-end data analysis project focuses on identifying factors that influence loan approval decisions using Excel and Power BI.

Kaggle Dataset Link: https://www.kaggle.com/datasets/architsharma01/loan-approval-prediction-dataset

## Project Overview

This dashboard analyzes loan application data to uncover patterns in approvals and rejections based on financial and demographic factors. The project includes data cleaning, transformation, KPI creation, and interactive dashboard development.

## Key Metrics

* **Total Loan Applications:** 4,241
* **Average CIBIL Score:** 599.69
* **Average Loan Amount:** 15,178,401
* **Approval Rate:** 62%
* **Rejection Rate:** 38%

## Analysis Highlights

* Loan approval distribution by:

  * Income group
  * Loan amount group
  * CIBIL score group

* Interactive filters for:

  * Self-employment status
  * Graduation status

* Loan term trend analysis in Power BI

## Tools & Technologies

* Microsoft Excel
* Power BI
* Power Query
* DAX

## Key Insights

* Higher CIBIL scores are strongly associated with increased loan approval rates.
* Applicants with lower income levels showed higher rejection percentages.
* Certain loan amount ranges had noticeably lower approval probabilities.

## Dashboard Features

* Interactive slicers 
* KPI cards for quick performance overview
* Comparative charts for approval analysis
* Clean and user-friendly dashboard design

The goal of this project is to demonstrate practical data analysis skills, dashboard design, and business insight generation using real-world financial data.
